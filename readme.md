# 📧 MailGlance – AI-Powered Email Summarizer

MailGlance is a full-stack AI-powered email management platform that integrates with Gmail to automatically summarize email threads using Large Language Models (LLMs). The system uses Google OAuth for secure authentication, Gmail APIs for email retrieval, Redis caching for improved performance, and OpenAI-powered NLP to generate concise, actionable summaries.

### Key Features

* AI-powered email thread summarization
* Gmail API integration
* Google OAuth 2.0 authentication
* Redis caching for faster responses
* Smart email search and filtering
* Responsive React dashboard
* Scalable Python backend

### Tech Stack

* Frontend: React.js
* Backend: Python
* AI/NLP: OpenAI API
* Authentication: Google OAuth 2.0
* Caching: Redis
* APIs: Gmail API

### Future Enhancements

* Email sentiment analysis
* Priority email detection
* Task extraction from emails
* Outlook integration
* AI-generated email replies

⭐ Built to improve productivity by transforming lengthy email conversations into concise, actionable insights.
